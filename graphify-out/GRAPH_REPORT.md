## How to read this

- **Edge direction:** `A -> B` means *A imports/depends on B*. So a module with high **in-degree** is depended upon by many others — load-bearing.
- **PageRank** runs on the import graph, so foundational modules (imported widely, directly or transitively) rank highest. Treat the top of the table as the **god nodes**: changing them has the widest blast radius.
- **Communities** are import-coupling clusters (label propagation). They tell a reviewer which modules move together.
- **Tested-by** counts the `tests/` files importing a module: load-bearing *and* well-tested vs load-bearing *and* thin.
